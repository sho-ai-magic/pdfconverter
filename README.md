# **PDF 多機能変換・編集ツール (PDF Converter Tool)**

ブラウザ上でPDFの並べ替え、編集（回転・黒塗り）、OCR文字認識、および各種形式（PNG, PPTX, GIF, PDF）への変換をすべて完結して行うことができるWebアプリケーションです。

サーバーにファイルをアップロードすることなく、すべての処理をローカルブラウザ上で行うため、プライバシーとセキュリティに優れています。

## **🚀 主な機能**

### **📝 PDF編集**

* **ページの並べ替え**: ドラッグ＆ドロップで直感的に順序を変更可能  
* **不要なページの削除**: ワンクリックで特定ページを除外  
* **ページ回転**: 時計回りに90度ずつ回転（プレビューと出力の両方に反映）  
* **黒塗り（マスキング）**: 機密情報などをドラッグ操作で塗りつぶし、画像として焼き付け

### **🔍 テキスト抽出（ハイブリッド方式）**

* **埋め込みテキスト抽出**: PDF内部のテキストデータを直接取得  
* **高精度OCR機能**: Tesseract.jsを使用し、スキャンされた画像PDFからも文字を認識（日本語・英語対応）

### **📥 多彩な出力形式**

* **PDF再構築**: 編集・加工内容を反映した新しいPDFを生成  
* **画像 (PNG)**: 全ページを画像化し、ZIP形式で一括保存  
* **PowerPoint (PPTX)**:  
  * 各ページをスライドとして保存  
  * **NotebookLMロゴマスク**: 右下の特定位置のロゴを隠す矩形を自動挿入  
  * **スピーカーノート連携**: 抽出したテキストをスライドノートに自動設定  
* **GIFアニメ**: PDFの各ページをパラパラ漫画のようなアニメーションとして保存

### **🎨 UI/UX**

* **モダンなデザイン**: Tailwind CSSを使用したクリーンなインターフェース  
* **リッチメニュー**: 保存・書き出し機能を右上の「保存・書き出し」ボタンに集約  
* **リアルタイム通知**: 処理完了やエラーをトースト通知でお知らせ

## **🛠 使用テクノロジー**

以下のオープンソースライブラリを使用しています。

* [Tailwind CSS](https://tailwindcss.com/) \- スタイリング  
* [PDF.js](https://mozilla.github.io/pdf.js/) \- PDFの解析とレンダリング  
* [PptxGenJS](https://gitbrent.github.io/PptxGenJS/) \- PowerPoint生成  
* [jsPDF](https://github.com/parallax/jsPDF) \- PDF生成  
* [Tesseract.js](https://tesseract.projectnaptha.com/) \- OCR（文字認識）  
* [JSZip](https://stuk.github.io/jszip/) \- ZIP圧縮  
* [GIF.js](https://jnordberg.github.io/gif.js/) \- GIFアニメーション生成  
* [SortableJS](https://sortablejs.github.io/Sortable/) \- ドラッグ＆ドロップ並べ替え

## **📂 使い方**

1. pdf\_converter.html をブラウザで開きます。  
2. 編集したいPDFファイルをドロップゾーンにドラッグ＆ドロップするか、ファイルを選択します。  
3. 必要に応じて「OCR（文字認識）を有効にする」にチェックを入れます。  
4. プレビュー画面で並べ替え、回転、黒塗り編集などを行います。  
5. 右上の「📥 保存・書き出し」ボタンから、希望の形式を選択してダウンロードします。
